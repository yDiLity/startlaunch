# AutoLaunch — Project Brief

## Цель проекта

AutoLaunch — кроссплатформенное десктопное приложение (Tauri), которое автоматически анализирует, настраивает и запускает проекты с GitHub. Пользователь вставляет ссылку на репозиторий — приложение делает всё остальное.

## Рамки проекта (Scope)

- Поддержка стеков: Node.js, Python, Rust, Go, Java, Docker, Static HTML
- Изоляция через Docker (sandbox mode) с fallback на прямой режим
- Сканер безопасности с предупреждениями
- Система снимков проектов (snapshots)
- Менеджер проектов с историей, тегами, поиском
- Система настроек с сохранением в файл
- Веб-интерфейс на React + TypeScript (Tauri frontend)
- Rust backend с SQLite

## Целевая аудитория

Начинающие разработчики, опытные инженеры, преподаватели, QA-инженеры.

## Ключевые метрики успеха

- Время от вставки ссылки до запуска < 5 минут для 90% типовых проектов
- Успешный запуск > 80% популярных репозиториев из Top-1000 GitHub
- Retention > 40% за 30 дней
