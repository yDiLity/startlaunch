## Интеграция с backend

ProcessWindow готов к интеграции с реальным backend:

1. **Прогресс**: Обновляется через `setProcessProgress()`
2. **Этапы**: Переключаются через `setProcessStage()`
3. **Логи**: Добавляются через `setProcessLogs(prev => [...prev, newLog])`
4. **Ошибки**: Устанавливаются через `setProcessError()` и `setProcessErrorSuggestion()`

Пример интеграции с Tauri:
```typescript
// Подписка на события от backend
listen('process-progress', (event) => {
  setProcessProgress(event.payload.progress);
  setProcessStage(event.payload.stage);
});

listen('process-log', (event) => {
  setProcessLogs(prev => [...prev, event.payload]);
});
```
