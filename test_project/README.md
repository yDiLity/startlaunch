# AutoLaunch Test Project

Простой тестовый проект для демонстрации возможностей AutoLaunch.

## Запуск

```bash
npm install
npm start
```

Сервер будет доступен на http://localhost:3000

## Что это демонстрирует

- Автоматическое определение Node.js проекта
- Парсинг package.json и извлечение команды запуска
- Установка зависимостей
- Запуск веб-сервера
- Автоматическое определение порта
